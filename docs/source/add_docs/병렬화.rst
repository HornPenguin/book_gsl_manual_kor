병렬화
***********************


디자인 문서에서 밝혔다시피 이 라이브러리는 병렬 처리를 고려하지 않습니다.
이 단원은 기초적인 병렬 컴퓨팅의 개요를 서술합니다. 
따라서 간결한 라이브러리 구현에서 병렬화를 왜 고려하지 않는지, 
어떠한 사항을 추가로 고려해야 하는지에 대해 간단히 알아봅시다.
