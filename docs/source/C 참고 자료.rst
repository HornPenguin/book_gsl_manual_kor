ASCII 코드표[#ASCII]_
----------------------------

ASCII는 정보 교환을 위한 미국 표준 문서표를 의미합니다[#ASCIICODE]_.
이 코드표는 7-bit 크기의 숫자로 이루어져 있으며 0-127까지의 숫자에 문자가 할당되어 있습니다.

DEC: 10진수
HX: 16진수
OCT: 8진수



확장된 ASCII는 128-255 숫자 범위를 추가로 가지며, 라틴 계열 문자에 대한 지원과 추가적인 선 기호, 수학 기호 등을 포함합니다.
